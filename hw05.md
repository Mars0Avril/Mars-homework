 | - | - | - |

 | 1 | 0 | 0 | 
 | 0 | 1 | 0 |
 | 1 | 1 | 1 |

 X = A*B

 2. 布尔表达式 Boolean Expression
 X = A⊕B

 3. 逻辑图 Logic Diagram Symbol
![](images/XOR逻辑图.png)
### AND意味着在两个输入电平不同的情况下输出高电平1，否则输出低电平0.
